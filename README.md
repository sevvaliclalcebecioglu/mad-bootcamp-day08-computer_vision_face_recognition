## üîπ Project Overview  

The projects follow a structured workflow covering both image and video-based computer vision tasks:

### Image & Video Handling
- Read and display images using OpenCV  
- Capture live video from webcam and IP cameras  
- Save images and videos from real-time streams  

### Image Processing Basics
- Convert images to grayscale  
- Split and merge RGB channels  
- Resize, blur, and threshold images  
- Visualize pixel-level image data  

### Edge & Feature Detection
- Sobel edge detection (horizontal & vertical edges)  
- Laplacian edge detection  
- Canny edge detection  
- Blob detection for object-like structures  

### Real-Time Applications
- Live sketch effect using edge detection  
- Motion detection using frame differencing  
- Selfie capture from webcam  
- Object tracking and video playback  

### Face Detection & Recognition
- Face detection using Haar Cascade classifiers  
- Eye detection within detected face regions  
- Dataset creation for face recognition  
- Face recognition using LBPH (Local Binary Patterns Histogram)  
- Real-time face recognition with confidence scoring
